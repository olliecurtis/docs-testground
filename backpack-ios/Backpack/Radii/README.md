# Backpack/Radii

## Usage

### Objective-C

```objective-c
#import <Backpack/Radii.h>

CGFloat myAppBorderRadiusSmall = BPKCornerRadiusSm;
CGFloat myAppBorderRadiusLarge = BPKCornerRadiusLg;
```

### Swift

```swift
import Backpack

let smallRadius = BPKCornerRadiusSm
let largeRadius = BPKCornerRadiusLg
```
